Import bank statement (PDF)

Text-based PDFs only (not scanned images). Results are AI-generated — always verify amounts against your statement before recording them anywhere.

AI setup: Google (recommended) or Vercel AI Gateway
  1. This site runs on Netlify (same as the expense form). Bank import calls a Netlify Function: /.netlify/functions/extract-bank-pdf.
  2. Option A — Google Gemini (no Vercel billing): create an API key in Google AI Studio. In Netlify → Environment variables, add GOOGLE_GENERATIVE_AI_API_KEY or GEMINI_API_KEY (scopes: Functions or All), then redeploy. Optional: GEMINI_MODEL (default gemini-2.5-flash).
  3. Option B — Vercel AI Gateway: open AI Gateway, create an API key, and set AI_GATEWAY_API_KEY in Netlify. Note: Vercel may require a payment method on file even to use included credits. Optional: AI_GATEWAY_MODEL (e.g. google/gemini-2.5-flash or other Gateway models).
  4. If both Google and Gateway keys are set, the function uses Google first.
  5. Leave the URL field below empty on Netlify; it uses the function path automatically. Override only for custom setups.
