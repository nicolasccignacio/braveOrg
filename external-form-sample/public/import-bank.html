Import bank statement (PDF)

Browser-only: PDF text is extracted with pdf.js here. Optionally add a Gemini API key for smarter parsing (sent from your browser to Google; may be blocked by CORS — then a simple line parser runs). Verify every row against your PDF.

How this page works
  1. No server or Netlify required — works on GitHub Pages.
  2. Text-based PDFs only (not scanned images).
  3. Optional Gemini key is stored in localStorage on this device. Treat it like a password; revoke the key in Google AI Studio if this machine is shared.

Optional: Gemini API

Leave the key empty to use heuristic parsing only. Values are saved in localStorage.
