Import bank statement (PDF)

Text-based PDFs only (not scanned images). Results are AI-generated — always verify amounts against your statement before recording them anywhere.

Vercel AI Gateway (one deploy)
  1. Import this repo in Vercel and set the project Root Directory to external-form-sample (same folder as public/ and api/).
  2. Open the AI Gateway tab in your Vercel team, create an API key if needed, and add AI_GATEWAY_API_KEY to the project (required for local vercel dev; production on Vercel can often use OIDC without the key — see AI Gateway authentication).
  3. Optional: set AI_GATEWAY_MODEL to another Gateway model id (default google/gemini-2.0-flash).
  4. Deploy. The site and API share one URL: this page calls /api/extract-bank-pdf automatically — no URL field needed.
  5. Only if you host these HTML files on Netlify (or elsewhere) but keep the API on Vercel: set ALLOWED_ORIGIN on Vercel to that site’s origin (e.g. https://yoursite.netlify.app), then paste your full Vercel API URL in the field below.
